的 particle

Starting with the easiest of the 3, 的 is commonly understood to be used in the context of possessing something. If you would use my, your, his/her in English, then you will most likely use 的.

- my name

- your view/opinion

As seen in the second example, a possession does not necessarily correspond to a physical object. You may also possess an opinion.

But 的 is also used when describing something:

- red apple

To think about this in English terms we could phrase it as "apple that is red". So in cases where you would use "object that is some attribute", you would use 的. Just keep in mind that the ordering of object and attribute is reversed in Chinese.

得 particle

In situations where you express the 'ability' to do something you would use 得. For example:

- Are you able to see it?

Here 得 is used to emphasize the ability to see. From my gathering it is almost equivalent to the same sentence without 得:

- Can you see it?

The second sentence feels more appropriate to use when you want to ask if the person is currently able to see something, while the first sentence asks if the person is able to see without him/her necessarily looking right now. The nuance is very small. The English translations I provided can also be considered to be synonymous while having some of that same nuance.

不 can be used with the opposite meaning of 得 to express the 'inability' to do something:

- Are you not able to see it?

地 particle

地 is used in situations where in English you would use the suffix -ly, such as simply or slowly:

- slowly

- He eats slowly
